Data Deletion Policy

Last Updated: June 25, 2025

1. Overview

This Data Deletion Policy explains how users can request deletion of their data from Instagram Bot Manager and what data is removed in the process.

2. What Data We Store

2.1 User Configuration Data

  • Bot settings and preferences
  • Business context from uploaded PDFs
  • Response personality and behavior settings
  • Platform connection preferences

2.2 API Credentials (Encrypted)

  • Instagram/Facebook access tokens
  • OpenAI API keys
  • Facebook Page IDs
  • Instagram Account IDs

2.3 Activity Logs

  • Bot response logs (30-day retention)
  • Error logs and debugging information
  • Usage statistics and performance metrics

2.4 Temporary Processing Data

  • Instagram comments (processed, not stored)
  • AI responses (logged temporarily for debugging)
  • Session data (cleared automatically)

3. How to Request Data Deletion

3.1 Immediate Self-Service Deletion

Option 1: Stop Using the Service

  1. Stop your bot in the Control tab
  2. Clear your credentials from the application
  3. Close the application
  4. Most session data is automatically cleared

Option 2: Revoke API Access

  1. Go to Facebook App settings
  2. Revoke permissions for the Instagram Bot app
  3. Go to OpenAI account settings
  4. Revoke or delete API keys
  5. This prevents further data access

3.2 Complete Data Deletion Request

For complete data removal, contact us:

Email: delete@instagrambot.example.com
Subject: "Data Deletion Request"

Include in your request:

  • Your Facebook Page ID or Instagram username
  • Email address used with the service
  • Confirmation that you want all data deleted
  • Preferred method of deletion confirmation

3.3 Automated Deletion Request

For Facebook compliance:

  • Submit deletion request through Facebook's Data Deletion API
  • Include your Facebook User ID or Page ID
  • Automatic processing within 30 days

4. Data Deletion Process

4.1 Deletion Timeline

  • Immediate: Session data and active connections
  • Within 24 hours: Configuration data and settings
  • Within 7 days: Encrypted API credentials
  • Within 30 days: All log files and historical data

4.2 What Gets Deleted

Immediately Removed:

  • ✅ Active bot sessions
  • ✅ Cached response data
  • ✅ Temporary processing files
  • ✅ Browser session storage

Within 24 Hours:

  • ✅ User configuration and settings
  • ✅ Uploaded business context PDFs
  • ✅ Personality and response preferences
  • ✅ Bot behavior customizations

Within 7 Days:

  • ✅ Encrypted API credentials
  • ✅ Access tokens and authentication data
  • ✅ Platform connection information
  • ✅ Account linking data

Within 30 Days:

  • ✅ Historical activity logs
  • ✅ Error logs and debugging information
  • ✅ Performance metrics and analytics
  • ✅ Backup copies in secure storage

4.3 What Cannot Be Deleted

Third-Party Data:

  • ❌ Instagram posts/comments (owned by Meta)
  • ❌ AI model training data (OpenAI's property)
  • ❌ Legal compliance records (retained as required)

Note: We cannot delete content from Instagram or OpenAI's systems, as this data is controlled by those platforms.

5. Verification and Confirmation

5.1 Identity Verification

Before processing deletion requests, we may verify:

  • Ownership of the Instagram/Facebook accounts
  • Email address used with the service
  • Additional verification for security purposes

5.2 Deletion Confirmation

We will confirm deletion by:

  • Email confirmation within 24 hours of completion
  • Deletion certificate if requested
  • Confirmation that third-party access has been revoked

6. Account Deactivation vs. Data Deletion

6.1 Account Deactivation

  • Stops bot activity but retains settings
  • Can be reactivated later
  • Data remains encrypted but accessible

6.2 Complete Data Deletion

  • Permanently removes all data
  • Cannot be undone or restored
  • Requires setting up service again from scratch

7. Automated Data Cleanup

7.1 Regular Cleanup

  • Session data: Cleared every 24 hours
  • Error logs: Deleted after 30 days
  • Temporary files: Removed after processing

7.2 Inactive Account Cleanup

  • Accounts inactive for 90 days: Configuration data removed
  • Accounts inactive for 180 days: All data deleted
  • Email notification sent before automated deletion

9. Regulatory Rights

Under applicable privacy laws (GDPR, CCPA, etc.), you have the right to:

  • Request deletion of personal data
  • Receive confirmation of deletion
  • File complaints with regulatory authorities
  • Receive data deletion certificates

For region-specific deletion procedures, please contact our privacy team.
